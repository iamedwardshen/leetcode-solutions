Arrays.

Coding Tips.

Use Arrays.fill or Collections.fill to fill.

boolean[] used = new boolean[10];
Arrays.fill(used, false);

Use Arrays.reverse or Collections.reverse to reverse.

int[] arr = {1, 2, 3, 4};
Arrays.reverse(arr); // {4, 3, 2, 1}

Use Arrays.sort or Collections.sort to sort.
